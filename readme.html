模式：Lv？提供經驗試算 → 模式：Lv？提供經驗試算

注意：模式：堆肥計算 → 當卡片等級超過1時
　　　模式：Lv？提供經驗試算
　　　這兩樣模式尚在測試中，警告訊息大致上有建立，
　　　但還是請自行注意輸入數值。
此工具於PTT的介紹(?)文： [工具] (肥料)卡片養成計算工具

○模式:堆肥計算 - 輸入卡片等級1
　同簡易說明，如果是餵異色卡得到的經驗就圈選異色

○模式:堆肥計算 - 當卡片等級超過1時
　同簡易說明請自行修改卡片等級並輸入對應等級經驗值。
　計算後餵食exp的部分會和輸入等級1時不同。
　這個計算主要是給轉蛋卡用的，
　所以會直接扣掉該卡目前等級累積的經驗值
　例：Rank S 卡片等級 40的 經驗值： 29114
　　　因為40剛好是最佳等級不用再升級，所以需餵食 0 exp
　如果是自行養成等級，有多餵經驗的話，
　可以用 卡片升級經驗表自行計算尚需經驗值
　另外狀態欄也會顯示"該卡餵給同色的初級經驗"
(卡片目前EXP可在卡片狀態中按左方寫著LV.及Next.的圈圈查看)

○模式：Lv？提供經驗試算
　總之先和堆肥計算共用格子。
　這個功能是計算某張卡在等級 ? 的時候可以提供多少經驗
　譬如說想知道
　Lv.1提供經驗值為 182 的Rank A卡，等級16 時會提供多少經驗值
　就輸入：
　Rank A 卡片等級 16 的經驗值： 182 　

關於成本倍率：
　　收到建議新增倍率~
　　成本為(投入經驗 + Lv.1經驗)，
　　Lv.1經驗可以拿去餵他卡所以視為成本
　　故(總獲得經驗值)/(LV.1獲得經驗值+付出經驗值)
