△使用說明△

模式：堆肥計算 → 輸入卡片等級1｜當卡片等級超過1時
模式：Lv？提供經驗試算 → 模式：Lv？提供經驗試算
→ 此計算機使用的推論、閒聊

注意：模式：堆肥計算 → 當卡片等級超過1時
　　　模式：Lv？提供經驗試算
　　　這兩樣模式尚在測試中，警告訊息大致上有建立，
　　　但還是請自行注意輸入數值。

○模式:堆肥計算 - 輸入卡片等級1
　同簡易說明，如果是餵異色卡得到的經驗就圈選異色

○模式:堆肥計算 - 當卡片等級超過1時
　請自行修改卡片等級。
　計算後餵食exp的部分會和輸入等級1時不同。
　這個計算主要是給轉蛋卡用的，
　所以會直接扣掉該卡目前等級累積的經驗值
　例如說輸入Lv.10 就是 只扣掉81，
　如果累積經驗值超過81的話可以利用 卡片升級經驗表自行計算
　另外狀態欄也會顯示"該卡餵給同色的初級經驗"

○模式：Lv？提供經驗試算
　總之先和堆肥計算共用格子。
　這個功能是計算某張卡在等級 ? 的時候可以提供多少經驗
　譬如說想知道
　Lv.1提供經驗值為 182 的Rank A卡，等級16 時會提供多少經驗值
　就輸入：
　Rank A 卡片等級 16 的經驗值： 182
